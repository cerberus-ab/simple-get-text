# simple-get-text
Simple gettext tool

### Installation
In browser:
```
<script src="gettext.min.js"></script>
```

Using npm:
```
$ npm i -g npm
$ npm i --save simple-get-text
```

In Node.js:
```
var GetText = require('gettext');
```
